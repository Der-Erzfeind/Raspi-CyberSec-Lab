\section*{Kurzdarstellung}
\label{sec:kurzdarstellung}
Diese Arbeit befasst sich mit der Entwicklung einer Testplatform für Cyberangriffe mit dem Fokus auf Funkverbindungen.
Im Rahmen den Projekts wurde auf Basis von RaspberryPi OS Software entwickelt, um Umgebungen zu schaffen, in denen verschiedene Cyberangriffe getestet werden können.



\section*{Abstract}
The goal of this project is to develop a flexible and expandible architecture, which can be used for demonstration and practice purposes in all kinds of IT applications.
In terms of features, the project focuses on the general user interface and device concept as well as testing environments for WiFi networks.
Therefore this thesis includes the documentation of the base program and hardware.
Also the basics of WiFi networks and their security, followed by testing of the system with some common cyber attacks.
Finishing, the worklog will lay out the development process and the problems encountered along development.
